Запустить проект
----------------------------
```bash
> npm install
> gulp
```

(!!! gulpfile.babel.js использует ECMAScript 6 код )

Или
----------------------------
в браузере открыть файл app/index.html.
если в браузере chrome, то в свойства браузера добавить
```bash
--allow-file-access-from-files --disable-web-security
```
